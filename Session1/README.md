### Hàm *print* trong Python

https://docs.python.org/3/library/functions.html#print

Hàm *print* nhận tham số đầu vào là chuỗi ký tự và in chuỗi ký tự ra output của chương trình

Ví dụ:

```python
print('ABCXYZ')
```

sẽ in ra

```shell
ABCXYZ
```
